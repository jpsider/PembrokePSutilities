# Invoke-Wait

## SYNOPSIS

## SYNTAX

```
Invoke-Wait [[-Seconds] <Int32>] [<CommonParameters>]
```

## DESCRIPTION
This function will pause for the specified amount of time.

## EXAMPLES

### EXAMPLE 1
```
Invoke-Wait -Seconds 15
```

## PARAMETERS

### -Seconds
A valid integer is optional.
Default is 10 seconds
